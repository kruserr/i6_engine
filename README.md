# i6_engine

Straight-Six Engine

<br>

# Deps

```
sudo dnf install glew-devel SDL2-devel SDL2_image-devel glm-devel freetype-devel
```

<br>

# Install

```
git clone https://github.com/kruserr/i6_engine.git
cd i6_engine
./run.sh

```
